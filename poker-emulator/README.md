# Poker Emulator

Эмулятор покерного стола для тестирования ботов и тренировки покерных ситуаций с использованием реальных скриншотов.

**Только фронтенд (HTML/CSS/JS) - никакого бэкенда не требуется!**

## Быстрый старт

```bash
# Просто откройте файл в браузере:
open poker-emulator/index.html
```

## Как это работает

**Прозрачные области клика**: Вместо рисования кнопок поверх скриншота, создаются **невидимые прозрачные области** поверх нативных кнопок на скриншоте.

Это даёт:
1. **Визуальную идентичность**: Бот видит оригинальные кнопки
2. **Точное тестирование**: Бот кликает по тем же координатам

## Файлы

- **`index.html`** - основное приложение для тестирования
- **`configurator.html`** - создание новых тестов (см. ниже)

## Создание нового теста (через configurator.html)

### Шаг 1: Откройте configurator.html

Просто откройте файл `configurator.html` в браузере.

### Шаг 2: Заполните информацию о тесте

```
Название теста (ID):      qq_4bet_pot
Название для отображения: QQ - 4бет пот
Описание:                 Играем 4бет пот с дамами
```

### Шаг 3: Настройте каждую улицу

1. **Выберите улицу** (Префлоп → Флоп → Турн → Ривер)
2. **Загрузите скриншот** для этой улицы
3. **Добавьте кнопки**:
   - Выберите тип (Fold/Check/Call/Bet/Raise)
   - Нажмите "Добавить"
   - Перетащите красную область на кнопку на скриншоте
4. **Укажите правильное действие** в выпадающем списке
5. **Добавьте объяснения** (опционально)
6. **Повторите для всех улиц**

### Шаг 4: Скачайте файлы

Нажмите кнопку **"Скачать файлы"**. Будут скачаны:
- `qq_4bet_pot_preflop.png`
- `qq_4bet_pot_flop.png`
- `qq_4bet_pot_turn.png`
- `qq_4bet_pot_river.png`
- `qq_4bet_pot_scenario.js` (код для scenarios.js)

### Шаг 5: Переместите файлы в проект

```bash
# 1. Переместите изображения
mv qq_4bet_pot_*.png poker-emulator/images/

# 2. Откройте qq_4bet_pot_scenario.js и скопируйте содержимое
# 3. Вставьте в poker-emulator/js/scenarios.js перед закрывающей скобкой SCENARIOS

# 4. Добавьте опцию в index.html (в select)
```

### Шаг 6: Готово!

Откройте `index.html` - новый тест появится в списке.

## API для ботов

### Элементы для взаимодействия

Каждая невидимая область имеет атрибуты:
- `id="area-{action}"` - ID области (например, `area-fold`)
- `data-action="{type}"` - тип действия (`fold`, `check`, `call`, `bet`, `raise`)
- `data-label="{text}"` - текст кнопки
- `data-amount="{amount}"` - сумма ставки

### Python + Selenium

```python
from selenium import webdriver
from selenium.webdriver.common.by import By

driver = webdriver.Chrome()
driver.get('file:///path/to/poker-emulator/index.html')

# Найти и кликнуть кнопку
fold_btn = driver.find_element(By.CSS_SELECTOR, '[data-action="fold"]')
fold_btn.click()

# Проверить результат
result = driver.find_element(By.ID, 'result-display')
print(result.text)  # "✓ Правильно!" или "✗ Неправильно..."
```

### Python + PyAutoGUI

import pyautogui

## Почему работает для CV

Для компьютерного зрения эмулятор **идентичен** реальному клиенту:

- ✅ Реальные скриншоты
- ✅ Оригинальные кнопки видны
- ✅ Нет элементов поверх
- ✅ Невидимые области прозрачны
- ✅ Координаты те же
