☁️ Test Creator (Cloud)

Создание тестов с изображениями по URL (GitHub Pages, CDN)

☁️ Облачный workflow:
  1. Загрузите скриншоты в интернет (GitHub Pages, imgur, etc.)
  2. Введите название теста
  3. Для каждой улицы укажите URL изображения
  4. Настройте кнопки: сайзы (33%, 50%, 75%, 100%), ползунок, действия (Fold/Check/Call/Bet/Raise)
  5. Скопируйте сгенерированный код в scenarios.js
  6. Закоммитьте изменения - тест сразу доступен онлайн!

💡 Подсказка: Правильное действие определяется автоматически из созданных кнопок. Бот должен нажать все созданные кнопки в последовательности: базовый сайз (если есть) → ползунок (если есть) → действие.
